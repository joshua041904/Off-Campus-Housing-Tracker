GOTCHA — aria-busy vs Playwright
--------------------------------
• webapp/app/listings/page.tsx uses aria-busy={loading} (boolean).
  In React, aria-busy={false} often OMITS the attribute from the DOM; Playwright
  toHaveAttribute("aria-busy", "false") then fails even when idle.
• Fix pattern: use string attributes, e.g. aria-busy={loading ? "true" : "false"}.
• Tests: webapp/e2e/listings-filters-maps.spec.ts, flows.spec.ts, guest.spec.ts
