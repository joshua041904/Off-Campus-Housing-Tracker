Off-Campus-Housing-Tracker: 7 Postgres DBs (5441–5447), backup & bring-up aligned (made with Tom)

- Postgres: Aligned with seven housing services. Host ports 5441–5447 (5432–5440
  reserved by another project). Mapping: auth 5441, listings 5442, bookings 5443,
  messaging 5444, notification 5445, trust 5446, analytics 5447.

- docker-compose.yml: Seven Postgres services — postgres-auth, postgres-listings,
  postgres-bookings, postgres-messaging, postgres-notification (new), postgres-trust,
  postgres-analytics. Removed legacy postgres, postgres-social, postgres-shopping,
  postgres-auction-monitor, postgres-python-ai. postgres-backup depends on
  postgres-auth and backs up DB auth.

- scripts/bring-up-external-infra.sh: Starts 7 Postgres services (no longer 8).
  Wait loop checks ports 5441–5447. RESTORE_BACKUP_DIR=latest uses backups/all-7-*.
  Comments and volume list updated for housing volumes only.

- scripts/backup-all-8-dbs.sh: Backs up 7 DBs on 5441–5447. PORT_DB map:
  auth, listings, bookings, messaging, notification, trust, analytics. Output
  directory backups/all-7-YYYYMMDD-HHMMSS. Version probe uses 5441/auth.
  Restore examples updated to 5441 and auth.

- scripts/backup-all-dbs.sh: New wrapper that runs backup-all-8-dbs.sh so
  callers can use either name for the same 7-DB backup.

- README.md: Postgres section lists 7 DBs (5441–5447) including
  notification-service; docker compose up command includes postgres-notification.

- Scripts (run-preflight-scale-and-all-suites.sh, ensure-ready-for-preflight.sh,
  check-all-pods-and-tls.sh): All references to 6 DBs / 5441–5446 updated to
  7 DBs / 5441–5447; docker compose up commands include postgres-notification;
  port wait loops and messages updated.
