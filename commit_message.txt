feat(analytics): versioned recommendation engine (models, weights, experiments) + proto

Database (analytics 5447)
------------------------
- infra/db/03-analytics-recommendation.sql: adds three tables for versioned ranking
  and traffic-split experiments.
  - recommendation_models: id, name, version, is_active, created_at. One baseline
    active at a time; multiple versions (e.g. baseline-ranking v1, geo-boost v2).
  - recommendation_weights: model_id (FK), distance, price, popularity, trust,
    recency. Per-model weights for hybrid scoring; no redeploy to tune.
  - recommendation_experiments: name, model_id (FK), traffic_percentage (0–100),
    is_active. Used with deterministic user bucketing (hash(user_id) % 100) to
    assign users to baseline vs experiment model.
- scripts/ensure-analytics-schema.sh: runs 03-analytics-recommendation.sql after
  02 (projections). Same port 5447, database analytics.
- infra/db/README.md: Analytics (5447) section updated to list
  recommendation_models / recommendation_weights / recommendation_experiments and
  the 03 script.

Proto (analytics + admin control plane)
---------------------------------------
- proto/analytics.proto:
  - import "common.proto" for Empty on admin RPCs.
  - AnalyticsService: added GetRecommendations(GetRecommendationsRequest) returns
    GetRecommendationsResponse. Request: user_id, latitude, longitude,
    radius_meters, min/max_price_cents. Response: repeated Recommendation
    (listing_id, score), model_name, model_version for observability and A/B.
  - RecommendationAdminService (internal): ActivateModel(name, version) returns
    Empty; SetExperimentTraffic(experiment_name, model_name, model_version,
    traffic_percentage) returns Empty. Enables rollback and traffic shifts
    without redeploy.
  - Messages: GetRecommendationsRequest, Recommendation, GetRecommendationsResponse,
    ActivateModelRequest, SetExperimentTrafficRequest.

No ML, no Redis cache, no async recompute in this change. Clean, deterministic,
versioned ranking with DB-controlled weights and experiment traffic.
